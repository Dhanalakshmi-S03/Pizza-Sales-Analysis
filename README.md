PROJECT TITLE:
Pizza Sales Performance Dashboard – Power BI
PROJECT OVERVIEW:
This Power BI dashboard provides a detailed analysis of pizza sales across different US branches. It helps understand sales trends, manager performance, pizza type popularity, and target achievement. The dashboard is fully interactive, enabling users to explore insights by state, manager, pizza type, and daily target metrics.
KEY METRICS:
•	Total Sales: ₹189.48K
•	Sales distribution by US states
•	Sales amount by manager
•	Total sales by type of pizza
•	Daily target vs total sales comparison
DASHBOARD FEATURES:
1.	Branch Map Visualization:
Shows sales contribution from different US states such as Arizona, California, Idaho, Nevada, and Oregon.
2.	Sales Amount by Manager:
A donut chart that highlights sales achieved by managers like Dan Peterson, Kyle Carr, Christy Olson, Mable Lindsey, and Rachel Gomez.
3.	Pizza Type Sales:
A bar chart displays total sales based on various pizza types such as Hot Pepperoni, Chicken Soup, BBQ Philly Steak, Veggie Feast, Margherita, and more.
4.	Target vs Sales (Daily):
A line + bar chart compares daily total sales with assigned targets.
5.	Interactive Filters:
Users can filter by state, manager, day, pizza type, and time range.
TOOLS USED:
•	Power BI Desktop
•	Excel (for raw data)
•	DAX for calculations and measures
